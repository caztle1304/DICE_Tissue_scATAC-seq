# R24_scATAC-seq
## Description
--------------
This repository contains scripts used to analyze scATAC-seq data

## Requirements 
--------------
- R(v4.0.1)
- ArchR(v1.0.1)
- Macs2(v2.2.7.1)
- Cell Ranger ATAC (v2.0.0)
## Raw data pre-processing 
--------------
- Pre-processing was done entirely using Cell Ranger ATAC (v2.0.0)
